# competitive-analysis

Framework for structuring a competitive analysis across dimensions: features, pricing, positioning, strengths, weaknesses.

## Requirements
- addCompetitor(analysis, { name, pricing, features, strengths, weaknesses })
- compareFeatureMatrix(analysis): returns feature-vs-competitor boolean grid
- rankByStrength(analysis): orders competitors by strength score
- renderReport(analysis): outputs markdown report with tables and summary
