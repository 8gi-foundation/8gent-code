# focus-trap

Focus trap logic for modal dialogs (DOM-free, index-based).

## Requirements
- FocusTrap with focusable element count
- next(currentIndex) returns next focusable index
- prev(currentIndex) returns previous focusable index
- first() and last() return boundary indices
- wrapAround enabled by default
