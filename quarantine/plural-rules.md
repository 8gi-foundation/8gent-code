# plural-rules

Pluralization rules for major languages.

## Requirements
- plural(count, locale) returns 'zero'|'one'|'two'|'few'|'many'|'other'
- Implements CLDR plural rules for en, fr, de, es, pt, ru, ar, zh
- selectForm(count, forms, locale) picks correct string from forms array
- Zero dependencies
