# csp-builder

Build Content-Security-Policy headers programmatically.

## Requirements
- CSPBuilder with directive methods: defaultSrc, scriptSrc, etc.
- nonce() generates a random nonce for inline scripts
- toString() generates header value string
- toMeta() generates <meta> tag content
- Zero dependencies
