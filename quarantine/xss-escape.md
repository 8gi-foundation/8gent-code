# xss-escape

Escape HTML special characters for safe output in HTML contexts.

## Requirements
- escape(str) replaces <, >, &, ", ' with entities
- escapeAttr(str) safe for HTML attribute values
- escapeCSS(str) safe for CSS string values
- escapeURL(str) safe for URL parameter values
- Zero dependencies
