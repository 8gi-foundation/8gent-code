# em-field-strength

Calculate electric and magnetic field strengths from point charges and current-carrying wires.

## Requirements
- Electric field from point charge via Coulomb's law
- Magnetic field from infinite wire via Biot-Savart
- Accept SI units throughout
- Return field magnitude and direction description
