# escape-velocity

Calculate escape velocity for any celestial body and orbital velocity at given altitude.

## Requirements
- v_escape = sqrt(2GM/r)
- Orbital velocity v_orbit = sqrt(GM/r)
- First, second, third cosmic velocities for Earth
- Table of escape velocities for solar system bodies
