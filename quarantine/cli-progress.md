# cli-progress

Terminal progress bar renderer (string-based, no cursor control).

## Requirements
- render(percent, width?, label?) returns bar string
- bar string: [####------] 45% label
- renderETA(percent, elapsed) appends ETA estimate
- style(filled, empty) customizes bar characters
- Zero dependencies
