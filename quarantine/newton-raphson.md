# newton-raphson

Find roots of functions using Newton-Raphson, bisection, and secant methods.

## Requirements
- Newton-Raphson with automatic numerical derivative
- Bisection method as fallback (guaranteed convergence)
- Secant method (derivative-free)
- Return root, iterations, and convergence flag
