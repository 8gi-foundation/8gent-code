# molecular-weight

Calculate molecular weight of a chemical formula by parsing element symbols and counts.

## Requirements
- Parse formulas like H2O, C6H12O6, Ca(OH)2
- Look up atomic masses from embedded periodic table
- Return molecular weight in g/mol
- Handle parentheses with multipliers
