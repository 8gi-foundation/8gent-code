# number-base-converter

Convert integers and fractions between bases 2-36 with arbitrary precision.

## Requirements
- Integer conversion any base to any base
- Fractional part conversion (up to configurable precision)
- Two's complement representation for negatives
- Named bases: binary, octal, decimal, hex
