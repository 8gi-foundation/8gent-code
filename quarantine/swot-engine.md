# swot-engine

SWOT analysis builder with scoring, prioritization, and strategy generation (SO, ST, WO, WT).

## Requirements
- addFactor(swot, quadrant, { text, score }): quadrant is S|W|O|T
- prioritize(swot): sorts each quadrant by score descending
- generateStrategies(swot): produces SO/ST/WO/WT strategy pairs
- renderMarkdown(swot): full SWOT report with prioritized factors and strategies
