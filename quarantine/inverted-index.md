# inverted-index

Build an inverted index for full-text search.

## Requirements
- InvertedIndex with add(id, text) and search(query)
- Tokenizes text to lowercase words
- search returns array of IDs sorted by frequency
- remove(id) removes document from index
- Zero dependencies
