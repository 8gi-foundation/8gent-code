# agent-scratchpad-v2

Structured agent scratchpad with typed entries: thoughts, plans, facts, and tasks.

## Requirements
- add(pad, type, content): type is thought | plan | fact | task | result
- facts(pad): returns all fact entries
- tasks(pad): returns pending task entries
- render(pad): formatted scratchpad view grouped by type
- summarize(pad): one-paragraph summary of pad contents
