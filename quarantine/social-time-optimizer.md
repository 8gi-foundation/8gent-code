# social-time-optimizer

Calculates optimal posting times per platform based on audience time zone and engagement patterns.

## Requirements
- optimalTimes(platform, timezone): returns top 3 posting windows
- platforms: twitter, linkedin, instagram, facebook, tiktok
- schedule(posts[], timezone): distributes posts across optimal windows
- renderSchedule(posts[]): formatted weekly posting calendar
