# path-normalize

Normalize and manipulate file paths cross-platform.

## Requirements
- normalize(path) collapses .., ., double slashes
- join(...segments) joins path segments
- relative(from, to) computes relative path
- isAbsolute(path) checks leading slash
- ext(path) returns file extension without dot
