# michaelis-menten

Fit enzyme kinetics data to Michaelis-Menten model and return Km, Vmax, and catalytic efficiency.

## Requirements
- v = Vmax*[S]/(Km+[S])
- Lineweaver-Burk linearization
- Competitive and non-competitive inhibition
- Return kcat from Vmax and [E_total]
