# Quarantine: Landing Page

## What
Static landing page for 8gent.dev - single HTML file, no build step.

## Files
- `apps/landing/index.html` - the landing page

## Brand compliance
- Colors: #E8610A orange accent, #F07A28 dark-mode variant, warm dark palette from BRAND.md
- Fonts: Fraunces 800 wordmark, Fraunces 700 headings, Inter body, JetBrains Mono code
- No em dashes, no purple/pink hues, no enthusiasm inflation
- Mobile-first with clamp() sizing, single-column below 600px
- Animated entrances via IntersectionObserver (0.4-0.5s staggered)

Ready for review. No existing files modified.
