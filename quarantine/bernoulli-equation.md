# bernoulli-equation

Apply Bernoulli's principle to calculate fluid velocity, pressure, and flow through constrictions.

## Requirements
- P + 0.5*rho*v^2 + rho*g*h = constant
- Venturi effect: velocity increase in constriction
- Torricelli's theorem: efflux velocity from tank
- Return pressure, velocity, and flow rate changes
