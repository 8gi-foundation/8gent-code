# big-int-math

BigInt arithmetic helpers for precision calculations.

## Requirements
- bigFloor(n, d) integer floor division with BigInt
- bigCeil(n, d) ceiling division
- bigGCD(a, b) greatest common divisor
- bigLCM(a, b) least common multiple
- bigPow(base, exp) integer exponentiation
