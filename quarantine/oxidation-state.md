# oxidation-state

Determine oxidation states for elements in compounds using electronegativity rules.

## Requirements
- Rule hierarchy: free element, monatomic ion, O=-2, H=+1, halogens
- Solve system of equations for unknown oxidation states
- Common polyatomic ions embedded
- Return oxidation state for each element
