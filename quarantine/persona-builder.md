# persona-builder

User persona builder from survey or behavioral data into structured persona cards.

## Requirements
- buildPersona({ name, age, role, goals[], pains[], motivations[], channels[] })
- renderCard(persona): formatted ASCII persona card
- comparePersonas(a, b): returns shared and divergent goals/pains
- exportJSON(persona): clean persona data structure
