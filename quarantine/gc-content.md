# gc-content

Calculate GC content, AT/GC ratio, melting temperature, and sequence complexity metrics.

## Requirements
- GC% = (G+C)/total * 100
- Tm (Wallace rule): 4*(G+C) + 2*(A+T) in Celsius
- Linguistic complexity (vocabulary/max-vocabulary)
- Dinucleotide frequency table
