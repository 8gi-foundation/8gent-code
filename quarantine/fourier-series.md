# fourier-series

Compute Fourier series coefficients (a_n, b_n) for periodic functions via numerical integration.

## Requirements
- Accept function and period as input
- Compute a0, a_n, b_n up to N harmonics
- Reconstruct function from coefficients
- Mean squared error between original and reconstruction
