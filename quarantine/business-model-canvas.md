# business-model-canvas

Business Model Canvas with all 9 blocks, validation, and markdown export.

## Requirements
- createCanvas(): returns empty canvas with all 9 blocks
- setBlock(canvas, block, items[]): block is one of the 9 BMC blocks
- validate(canvas): returns list of empty or weak blocks
- renderMarkdown(canvas): formatted canvas as a readable document
- scoreCompleteness(canvas): 0-100 score
