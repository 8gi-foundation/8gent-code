# ugc-prompt-builder

User-generated content prompt builder for community engagement and social proof collection.

## Requirements
- buildPrompt({ type, brand, product, platform, incentive? }): returns UGC prompt
- types: review, photo, video, story, testimonial
- moderate(text): checks for prohibited content patterns
- toCampaignBrief(prompts[]): full UGC campaign brief document
- renderBatch(prompts[]): formatted prompt collection
