# stellar-luminosity

Calculate stellar luminosity, radius, and temperature relationships using HR diagram parameters.

## Requirements
- L = 4*pi*R^2*sigma*T^4
- Main sequence mass-luminosity relation L proportional to M^3.5
- Stefan-Boltzmann: radius from L and T
- Spectral class to temperature mapping
