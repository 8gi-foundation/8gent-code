# projectile-motion

Calculate range, max height, and time of flight for projectile motion given initial velocity and angle.

## Requirements
- Accept velocity (m/s) and angle (degrees)
- Return range, apex height, flight time, and velocity components
- Support optional initial height above ground
- Use standard gravity 9.81 m/s^2 with override option
