# screen-reader-text

Generate screen-reader-friendly text alternatives.

## Requirements
- describe(node) returns verbose description from node metadata
- abbreviate(text, expansions) expands known abbreviations
- tableCaption(rows, cols) describes table structure
- progressText(value, max, label?) returns '75% complete'
- Zero dependencies
