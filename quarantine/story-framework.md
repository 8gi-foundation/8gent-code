# story-framework

Brand storytelling framework builder (Hero's Journey, Problem-Agitate-Solve, STAR).

## Requirements
- heroJourney(elements{}): builds narrative across 12 stages
- pas({ problem, agitation, solution }): Problem-Agitate-Solve framework
- star({ situation, task, action, result }): STAR story format
- aida({ attention, interest, desire, action }): AIDA marketing framework
- renderStory(framework, type): formatted story document
