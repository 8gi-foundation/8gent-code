# go-to-market-planner

GTM strategy builder with ICP definition, channels, messaging pillars, and 90-day milestone plan.

## Requirements
- defineICP({ company, role, pain, gain, trigger })
- addChannel(gtm, { name, type, audience, budget, kpi })
- buildMessagingPillars(gtm, pillars[]): value propositions with proof points
- generate90DayPlan(gtm): phased milestone plan with owners
- renderDocument(gtm): full GTM strategy document in markdown
