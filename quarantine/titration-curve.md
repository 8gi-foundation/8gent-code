# titration-curve

Generate titration curve data points for strong-strong, weak-strong, and polyprotic acid-base titrations.

## Requirements
- Strong acid + strong base: exact solution
- Weak acid + strong base: buffer region + equivalence point
- Return array of {volumeAdded, pH} data points
- Identify equivalence point volume and pH
