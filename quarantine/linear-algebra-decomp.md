# linear-algebra-decomp

Perform LU, QR, and Cholesky decompositions on matrices.

## Requirements
- LU decomposition with partial pivoting
- QR via Gram-Schmidt orthogonalization
- Cholesky for symmetric positive-definite matrices
- Solve Ax=b using each decomposition
