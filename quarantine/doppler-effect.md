# doppler-effect

Calculate observed frequency due to Doppler effect for moving source or observer.

## Requirements
- Accept source frequency, source velocity, observer velocity
- Support sound (user-supplied medium speed) and light (relativistic) modes
- Return observed frequency and shift ratio
- Handle edge cases: source moving away vs toward
