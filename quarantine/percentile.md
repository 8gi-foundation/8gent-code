# percentile

Compute percentiles and quantiles from sorted arrays.

## Requirements
- percentile(sorted, p) returns value at percentile p (0-100)
- quantile(sorted, q) returns value at quantile q (0-1)
- iqr(sorted) interquartile range
- quartiles(sorted) returns Q1, median, Q3
- Zero dependencies
