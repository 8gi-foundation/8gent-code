# circuit-resistance

Compute total resistance for series and parallel resistor networks.

## Requirements
- seriesResistance(resistors[]) returns sum
- parallelResistance(resistors[]) returns reciprocal-sum inverse
- combinedNetwork(spec) parses nested series/parallel object
- Return total resistance in ohms
