# unit-convert

Convert common physical units.

## Requirements
- convert(value, from, to) returns converted value
- Supports: length (m, km, ft, in, mi), mass (kg, g, lb, oz), temp (C, F, K)
- volume (l, ml, gal, fl oz)
- speed (km/h, mph, m/s)
- Zero dependencies
