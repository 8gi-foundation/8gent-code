# aria-label

Generate accessible ARIA labels from element context.

## Requirements
- labelFor(element) returns descriptive label string
- buttonLabel(text, icon?) combines text and icon context
- inputLabel(name, type, required) builds form input label
- listLabel(count, noun) builds 'N items' style label
- Zero dependencies
