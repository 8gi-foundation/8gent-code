# modular-exponentiation

Compute modular exponentiation and solve modular equations using fast repeated squaring.

## Requirements
- modPow(base, exp, mod) in O(log exp)
- Handle BigInt for arbitrary precision
- Modular inverse (Fermat's little theorem when mod is prime)
- Chinese Remainder Theorem solver
