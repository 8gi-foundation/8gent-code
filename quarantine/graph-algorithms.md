# graph-algorithms

Run Dijkstra, BFS, DFS, and topological sort on weighted directed/undirected graphs.

## Requirements
- Accept adjacency list or matrix representation
- Dijkstra single-source shortest paths
- BFS and DFS with traversal order output
- Connected components and cycle detection
