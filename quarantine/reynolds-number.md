# reynolds-number

Compute Reynolds number and classify laminar vs turbulent flow in pipes and over flat plates.

## Requirements
- Accept fluid density, velocity, characteristic length, dynamic viscosity
- Return Re value and flow regime classification
- Support kinematic viscosity as input alternative
- Include common fluid presets (water, air at STP)
