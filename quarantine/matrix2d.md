# matrix2d

2D transformation matrix for graphics.

## Requirements
- Matrix2D: a, b, c, d, tx, ty
- multiply(m1, m2) matrix multiplication
- translate(tx, ty), rotate(angle), scale(sx, sy) factory methods
- invert(m) returns inverse
- transformPoint(m, x, y) applies transform
