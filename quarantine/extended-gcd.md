# extended-gcd

Compute GCD, LCM, and Bezout coefficients using the extended Euclidean algorithm.

## Requirements
- extendedGcd(a,b) returns {gcd, x, y} where ax+by=gcd
- LCM from GCD
- Modular inverse from Bezout (when gcd=1)
- Handle BigInt inputs for large numbers
