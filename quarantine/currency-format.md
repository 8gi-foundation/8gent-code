# currency-format

Format currency values by locale and currency code.

## Requirements
- format(amount, currency, locale) returns formatted string
- Handles major currencies: USD, EUR, GBP, JPY, BRL, etc.
- Compact notation: 1.2K, 3.4M
- Negative formatting with parentheses option
- Zero dependencies
