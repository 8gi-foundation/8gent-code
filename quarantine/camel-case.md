# camel-case

Convert strings between camelCase, PascalCase, snake_case, kebab-case, and SCREAMING_SNAKE_CASE.

## Requirements
- Export toCamel, toPascal, toSnake, toKebab, toScreaming
- Handle mixed input correctly (e.g. 'foo-barBaz' -> 'fooBarBaz')
- Preserve consecutive uppercase acronyms (HTTP -> http)
- Zero dependencies
