# base-convert

Convert numbers between arbitrary bases 2-36.

## Requirements
- convert(n, fromBase, toBase) converts string representation
- toBinary(n) shortcut for base 2
- toHexStr(n) shortcut for base 16
- toOctal(n) shortcut for base 8
- Zero dependencies
