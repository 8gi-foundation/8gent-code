# entropy

Estimate entropy of a string for password strength analysis.

## Requirements
- entropy(str) returns bits of entropy estimate
- charsetSize(str) returns character set cardinality
- zxcvbn-lite(str) returns score 0-4 based on patterns
- isCommonPassword(str) checks against small blocklist
- Zero dependencies
