# segment-tree

Segment tree for range sum queries and point updates.

## Requirements
- SegmentTree(arr) builds from array
- query(l, r) returns sum of range
- update(i, value) updates element
- Supports sum queries (extend for min/max by swapping fn)
- Zero dependencies
