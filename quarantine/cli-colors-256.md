# cli-colors-256

256-color and truecolor ANSI escape code generator.

## Requirements
- fg256(code) returns ANSI escape for 256-color fg
- bg256(code) returns ANSI escape for 256-color bg
- fgRGB(r, g, b) truecolor fg
- bgRGB(r, g, b) truecolor bg
- strip(str) removes all ANSI codes
