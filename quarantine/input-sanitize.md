# input-sanitize

Strip dangerous characters from user-supplied input strings.

## Requirements
- sanitize(input) removes control characters and null bytes
- stripHTML(input) removes HTML tags
- normalizeWhitespace(input) collapses runs of whitespace
- maxLength(input, limit) truncates safely
- Zero dependencies
