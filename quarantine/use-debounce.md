# use-debounce

React hook for debouncing values and callbacks.

## Requirements
- useDebounce<T>(value, delay) returns debounced value
- useDebouncedCallback(fn, delay) returns stable debounced fn
- Clears on unmount
- Works with React 18+
- Zero dependencies beyond React
