# landing-page-copywriter

Structured landing page copy builder: hero, social proof, features, objections, CTA sections.

## Requirements
- buildPage({ product, audience, primaryCTA, pain, gain })
- heroSection(page): headline, subheadline, CTA button copy
- featuresSection(page, features[]): benefit-first feature descriptions
- objectionHandler(objections[]): FAQ-style objection responses
- renderMarkdown(page): full landing page copy document
