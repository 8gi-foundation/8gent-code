# regex-escape

Escape strings for use in regular expressions.

## Requirements
- escape(str) escapes all regex special characters
- escapeReplacement(str) escapes $ in replacement strings
- toPattern(str, flags?) builds RegExp from literal string
- isRegex(value) type guard
- Zero dependencies
