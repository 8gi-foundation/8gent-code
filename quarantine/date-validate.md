# date-validate

Validate and parse date strings in multiple formats.

## Requirements
- isValid(str) returns boolean
- parse(str) returns Date or null
- format: ISO 8601, DD/MM/YYYY, MM/DD/YYYY, YYYY.MM.DD
- isInRange(date, from, to) checks bounds
- isFuture(date) and isPast(date) helpers
