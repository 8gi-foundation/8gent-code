# assert

Rich assertion library for tests and runtime contracts.

## Requirements
- assert(condition, message?) throws AssertionError
- assertEqual(a, b), assertNotEqual(a, b)
- assertDeepEqual(a, b), assertThrows(fn)
- assertRejects(asyncFn) -> Promise
- AssertionError includes actual and expected values
