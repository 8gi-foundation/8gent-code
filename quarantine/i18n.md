# Quarantine: i18n Utility

**Package:** `packages/tools/i18n.ts`
**Status:** Quarantined - not wired into any consumer yet
**Size:** ~110 lines, zero external dependencies

## What It Does

Internationalization utility for 8gent. Loads JSON translation files, resolves keys with dot-notation, interpolates variables, and handles pluralization.

## API

```ts
import { I18n } from "@8gent/tools/i18n";

const i18n = new I18n({
  localesDir: "./locales",  // directory with en.json, pt-BR.json, etc.
  fallback: "en",           // default fallback
  locale: "pt-BR",          // override auto-detection
});

// Simple lookup
i18n.t("greeting"); // "Hello"

// Interpolation
i18n.t("welcome", { name: "James" }); // "Welcome, James"

// Pluralization
i18n.t("items", { count: 1 }); // "1 item"
i18n.t("items", { count: 5 }); // "5 items"

// Dot-notation keys
i18n.t("errors.notFound"); // "Not found"

// Runtime locale switch
i18n.setLocale("ja");
```

## Translation File Format

```json
{
  "greeting": "Hello",
  "welcome": "Welcome, {{name}}",
  "items": {
    "zero": "No items",
    "one": "1 item",
    "other": "{{count}} items"
  },
  "errors": {
    "notFound": "Not found"
  }
}
```

## Locale Detection

Reads `LANG`, `LC_ALL`, or `LANGUAGE` from the environment. Falls back to `"en"`.
