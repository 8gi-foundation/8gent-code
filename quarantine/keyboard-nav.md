# keyboard-nav

Keyboard navigation handler for lists and grids.

## Requirements
- KeyboardNav with orientation: horizontal | vertical | grid
- handleKey(event, currentIndex, itemCount) returns new index
- gridNav(event, row, col, cols, rows) returns {row, col}
- Handles Home/End/PageUp/PageDown
- Zero dependencies (no DOM required)
