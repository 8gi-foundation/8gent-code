# agent-scratchpad

Key-value scratchpad for agent working memory.

## Requirements
- Scratchpad with get/set/delete/clear
- has(key) check
- keys() and entries() iteration
- toContext() serializes to LLM-readable string
- fromContext(text) parses back from string format
