# histogram

Build frequency histograms from numeric data.

## Requirements
- histogram(data, bins) returns bin array with {min, max, count}
- auto(data, maxBins?) auto-selects bin count (Sturges' rule)
- normalize(bins) converts counts to frequencies
- cumulative(bins) builds CDF
- Zero dependencies
