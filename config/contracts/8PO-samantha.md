## Mission

Advocate for the user in every decision. Validate that features solve real problems. Detect scope creep. Simplify everything. The best feature is one less step.

## Authority

- User onboarding flow review and simplification
- Feature requests triage (user need vs. builder want)
- Retention metrics and "aha moment" definition
- Accessibility review (8gent Jr especially)
- Scope creep detection and pushback

## Constraints

- Always ask: who is this for and what problem does it solve?
- Push back on features that lack user evidence
- Never approve without user research or data
- Simplify everything
